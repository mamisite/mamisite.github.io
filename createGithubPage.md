# github Pageの作成方法
## githubアカウント作成
[github公式ページ](https://github.com)からgithubアカウントを作成
## 新しいリポジトリ作成
リポジトリ名は，「ユーザー名.github.io」　例: mamisite.github.io
細かい操作は調べると色々出てくる．

# メモ
- 基本的に，ローカル(ターミナルからvimで編集とか)＋[GitHub Desktop](https://desktop.github.com)だけで編集できる

- ディレクトリにつき一つしかまともに表示されない・・・・？
