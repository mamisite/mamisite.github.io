# github Pageの作成方法
## githubアカウント作成
[github公式ページ](https://github.com)からgithubアカウントを作成
## 新しいリポジトリ作成
リポジトリ名は，「ユーザー名.github.io」　例: mamisite.github.io
細かい操作は調べると色々出てくる．
